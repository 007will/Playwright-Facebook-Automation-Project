# Playwright Facebook Automation Project

![CI](https://github.com/007will/Playwright-Facebook-Automation-Project/actions/workflows/ci.yml/badge.svg)

Automated UI testing project built with **Playwright**, focused on cross-browser flows and robust UI automation patterns. Includes examples of fixtures, traces, screenshots and reusable helpers for resilient tests.

Quickstart
- Install: `npm ci`
- Run headless: `npm run test:ci` (if present)

What to review
- Tests organized by feature and cross-browser examples.
- Evidence: screenshots, traces and artefacts on CI.
- CI workflow executes headless runs and uploads artifacts.
